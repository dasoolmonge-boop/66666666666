# 📜 Технический паспорт проекта ООО «Чалама»

## 1. Общая информация
**Тип проекта**: Система онлайн-бронирования для гостиничного комплекса (Отель «Чалама») и загородного комплекса (Юрты «Хаан-Дыт»).
**Стек**: JavaScript (Node.js/Express) + SQLite (база данных) + Vanilla HTML/CSS (фронтенд).
**Адаптивность**:
- `index.html` — основная версия для ПК.
- `mobile.html` — оптимизированная версия для мобильных устройств (открывается при заходе из Telegram-бота или с телефона).

## 5. Структура системы
- **Бэкенд**: `backend/index.js` (управляет API, базой данных и уведомлениями).
- **База данных**: `/root/chalama_project/backend/db/database.sqlite`
    - Таблица `bookings` — все заказы.
    - Таблица `rooms` — список номеров, юрт и цен.
    - Таблица `admins` — Telegram ID администраторов, которым приходят уведомления.
- **Процессы (PM2)**:
    - `chalama-backend` (ID 0) — сервер API.
    - `chalama-bot` (ID 1) — Telegram бот.

## 6. Текущий статус
