# Хроника разработки проекта «ООО Чалама»

В данном документе собрана вся история взаимодействия, принятых решений и реализованных функций по проекту создания цифровой экосистемы для отеля «Чалама» и сопутствующих направлений.

---

## 📅 Сессия 1: Оптимизация бота и логики бронирования (9443817f)
**Цель**: Изменение пользовательского пути в Telegram Mini App для более интуитивного бронирования юрт.

### Ключевые изменения:
- **Инверсия выбора**: Изменен поток с "Выбор юрты -> Выбор дат" на "Выбор дат -> Фильтрация доступных юрт".
- **Динамическая доступность**: Реализована логика на фронтенде, которая запрашивает все существующие бронирования с сервера и скрывает занятые юрты на выбранные даты.
- **Интеграция**: Настройка `cors` и `dotenv` для корректной работы серверной части на хостинге.

---

## 📅 Сессия 2: Подготовка к деплою и управление структурой (444add37)
**Цель**: Реорганизация файлов проекта для успешного запуска на платформе Bothost.ru.

### Ключевые изменения:
- **Реструктуризация**: Файлы `package.json` и `.env` перенесены в корень проекта для правильной работы системы авто-деплоя.
- **Конфигурация**: Прописаны боевые переменные (токен бота, ID администратора, домен API).
- **Очистка**: Удалены лишние файлы шаблонов (например, `main.py`), чтобы избежать конфликтов распознавания языка программирования на хостинге.

---

## 📅 Сессия 3: Глобальное обновление - Сайт и Админ-панель (5dcdc2d0)
**Цель**: Создание современного полнофункционального веб-сайта с базой данных, админ-панелью и продвинутым UI.

### Ключевые изменения:
- **Технологический стек**: Переход на **Next.js 14** (App Router), **Prisma ORM** (SQLite) и **TailwindCSS**.
- **Дизайн-система**: Реализован премиальный стиль с использованием эффектов стеклянного морфизма (glassmorphism), золотых акцентов и плавных анимаций (`framer-motion`).
- **Функционал**:
    - Страницы бронирования номеров и юрт с расчетом цены.
    - **Админ-панель**: Полноценный Dashboard с графиками, управлением бронированиями (подтверждение/отмена) и CRUD для номеров/юрт.
    - **OCR (Распознавание текста)**: Интеграция `tesseract.js` для автоматического извлечения цен из загруженных изображений прайс-листов.
- **Контент**: Генерация уникальных HD-изображений интерьеров и архитектуры через ИИ-инструменты.

## 📅 Сессия 4: Консолидация и Технический паспорт (06.04.2026)
**Цель**: Официальное закрепление структуры проекта, доступов и текущего стека.

### Ключевые изменения:
- **Упрощение стека**: Официально подтвержден переход с Next.js на **Node.js/Express + Vanilla HTML/CSS**.
- **Адаптивность**: Разделение на `index.html` (ПК) и `mobile.html` (Mobile) для точного контроля UI.
- **Интеграция уведомлений**: Подключена платформа **MAX** для уведомлений администраторов (токен `f9LH...`).
- **Бронирование**: Настроено три независимых потока: Отель, Юрты «Хаан-Дыт» и Сауна.

## 🏛️ Текущая архитектура (Актуально на 06.04.2026)
- **Фронтенд**: Vanilla HTML5 / CSS3 / Modern JS.
  - `backend/public/index.html` (Desktop)
  - `backend/public/mobile.html` (Mobile)
- **Бэкенд**: Node.js + Express.
  - Основной файл: `backend/index.js`
- **БД**: SQLite (`backend/db/database.sqlite`).
  - Таблицы: `bookings`, `rooms`, `admins`.
- **Инструменты**: 
  - **PM2**: `chalama-backend`, `chalama-bot`.
  - **MAX API**: Для системных уведомлений.
  - **Telegram Bot**: Интерфейс для пользователей и админов.
